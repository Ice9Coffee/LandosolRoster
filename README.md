# LandosolRoster
兰德索尔花名册

记录Princess Connect Re:Dive中角色的各种昵称、别称、简称，  
使竞技场作业查询、猜角色小游戏等HoshinoBot插件更易使用！

### 如何贡献新别称？
- 提交issue或pr
- ~~使用HoshinoBot插件（开发中）~~

#### 注意！以下别称将大概率不会被收录！
- 含有对角色侮辱性称呼的，如：~~安: 坦克~~
- 含有vtuber/vup梗的，如：~~真步(夏日): 水嘉然~~
- 含有跨作品梗的，如：~~流夏(夏日)：亚索~~
  - 例外：若别称来源为**广泛传播**的**经典**/**二次元**作品，将会考虑收录，如：贪吃佩可(公主): 高达
- 长度大于官方名称的，如：~~真琴: 伤害优衣的家伙~~
- 可能同时代表多个角色的，如：~~真琴: 伤害优衣的家伙~~
